# Ejectuar en desarrollo

1. Clonar en repositorio
2. Ejecutar
```
yarn install
```
3. Tener nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```

## Stack usado
* MongoDB
* Nest
